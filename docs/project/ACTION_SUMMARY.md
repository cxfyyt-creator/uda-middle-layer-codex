## 关键结论

1. **系统已是 IR**，不需从零设计，需从"案例型"升级到"工程型"
2. **井/调度解析是当前最大短板**，直接影响 CMG 链路可用性
3. **角点网格和特殊模型**是能力边界，短期内不应作为支持目标
4. **回归测试缺失**是质量风险，需尽快补齐
